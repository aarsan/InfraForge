Key Differentiators

  • CISO Agent — IT defines security policies through chat, not code
  • AI Onboarding — AI writes, tests, and versions ARM/Bicep templates
  • Template Composition — Build landing zones from onboarded services
  • Zero Code — No IaC authoring, no policy manifests, no pipeline YAML
  • ARM SDK Native — Direct Azure deployment with What-If preview
  • Enterprise Auth — Entra ID SSO with identity-aware tagging

Tech Stack

  • Runtime: Python 3.9+ / FastAPI
  • AI: GitHub Copilot SDK (GPT-4.1, Claude Sonnet)
  • Data: Azure SQL (catalog, governance, standards)
  • Analytics: Fabric IQ (OneLake sync → Power BI)
  • Auth: Entra ID App Registration (OAuth2 + MSAL)
  • Identity: Microsoft Work IQ (MCP server — M365 org intelligence)
  • Deploy: ARM SDK (no az/terraform/bicep CLI needed)
  • Frontend: Vanilla JS SPA with WebSocket streaming
